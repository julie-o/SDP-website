Helping your visitors find their way

GuidEd is designed to lead visually impaired users and those unfamiliar with a location around, using voice commands or the attached touch screen.

Learn More

Using GuidEd.

Using GuidEd is simple, just pick a location by using voice commands or the touch interface, and follow the robot to your final location. The robot is equipped with a comfortable handle to hold on to, but you can alternatively just follow the robot without holding on to it.

Touch interface and voice commands.

You can use GuidEd either by touch screen interface or voice commands. The intuitive design lets you choose a room from a set list of goals, and then the journey begins. Using voice commands works just as pressing a button; state the room you want to go to and head off!

Technical details.

The robot base is a Turtlebot equipped with a Rasberry Pi and Lidar, ideal for moving and navigating in all directions. The user interface and audio equipment is connected using a Raspberry Pi, which communicates with the Pi on the Turtlebot via a DICE machine (the DICE machine acts as a server, so that the Raspberry Pi does not need to do as many calculations).
